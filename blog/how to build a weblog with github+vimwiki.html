How to setup a personal weblog with github and vimwiki

2012/08/12 20:28:50 begin

1. register a github

  1. You should register your github account from http://www.github.com .
  2. Create a repo named yourname.github.com ( be careful, it MUST BE named yourname.github.com, or you cannot visit your page using yourname.github.com later.)
  3. get a clone from github. revise and upload using git pull git commit git push or something like this. Or you can write a shell script to upload code.

2. prepare for vimwiki

  1. You should have a vim in Linux MacOS or Windows. :)
  2. Download vimwiki script from http://www.vim.org/scripts/script.php?script_id=2226
  3. config your personal vimrc. Locate your vimwiki path to the github project directory. ( I am not recommand you also to locate your vimwiki source path to the project directory. )
  4. write your own css file, and config template path in vimrc if you need.
  5. happy your weblog.

3. Tips cannot search from Google

  1. In current version of vimwiki (version 2.0.1) , using template_path template_default template_ext instead header_path and footer_path.
  2. [Link Description] no longer supported in version 2.0.1 of vimwiki. I don't think it is a smart decision. Now I cannot find a suitable way to reslove it.
  3. DONOT upload your personal dairy to github server :).

4. Related information

  1. Introduce of vimwiki (Chinese): http://wiki.ktmud.com/tips/vim/vimwiki-guide.html
  2. Writing in vimwiki and github (Chinese): http://blog.hotoo.me/use-vimwiki-for-blog.html
  3. Git help: https://help.github.com/
  4. Git Pages help: http://pages.github.com/
