basic summation formulas

  1. 12+22+...+n2=n(n+1)(n+2)/6
  2. 13+23+...+n3=n2(n+1)2/4
  3. x+2x2+3x3+...+nxn=x/(1-x)2 if and only if |x|<1
  4. 1+x+x2+...+xn=(1-xn)/(1-x) if and only if |x|<1, = 1/(1-x)
